# Intellij Shortcuts

- ### Search

  + ##### Search class or methods or functions
    > **Shift** **x 2**
  
  + ##### Search class
    > **Ctrl** + **N**  
  
  + ##### Search method or function
    > **Ctrl** + **F12**  
    > **cmd** + **F12** 
  
  + ##### Search method or function can be override
    > **Ctrl** + **o**
